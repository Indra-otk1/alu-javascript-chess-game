# Frontend Preservation Notes

The current board presentation is part of the project constraints and should remain visually unchanged during the API integration work.

Elements to preserve:
- The existing chessboard grid structure in `source code/index.html`
- The current board and square styling in `source code/style.css`
- The current hover and highlight visual language
- The overall page layout centered around the board

Allowed additions:
- Server integration code
- Minimal functional text or status content below the board
- Minimal controls needed for gameplay, such as reset, as long as they do not redesign the page
